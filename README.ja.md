# 📜 Covenant (カヴェナント)

**Binding Agreements. Clear Responsibilities.**

## 📖 概要 (Overview)
**Covenant** は、会議の議事録を単なる「記録」から「契約」へと昇華させるタスク管理・議事録AIです。
「誰が」「いつまでに」「何をするか」を明確にし、曖昧な責任によるタスク漏れ（空中分解）を完全に防ぎます。

## ✨ コンセプト (Concept)
*   **Auto-Start:** 話し始めると自動で録音開始 (VAD搭載)。
*   **Action Driven:** 議事録から「誰が」「いつまでに」「何をやるか」を自動抽出。
*   **Auto-Save:** 分析結果をSupabaseへ自動保存。
*   **Binding:** 合意形成の瞬間を逃さない。

## 🛠 技術スタック (Tech Stack)
*   **Input:** Gemini 2.0 Flash Exp (Audio Processing)
*   **Analysis:** Gemini 2.0 Flash Exp
*   **Integration:** Slack / Notion / Calendar
